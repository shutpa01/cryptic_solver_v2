## Session 2026-04-09: UX Overhaul + Tutorial Puzzle Design

### Completed UX fixes
- **Hidden word display**: `understo ODOURS on's` format — answer separated from fodder in uppercase. Updated in 6 files (report.py, backfill_dd_hidden.py, batch_signature_solve.py, admin.py, learn.py, explanation_builder.py)
- **Explanation delivery**: Removed auto-shown blog explanations and Generate Explanation button. Explanations now only served behind progressive hint buttons. Blog explanations served as fallback with attribution link (blog_name + URL in hint_step.html)
- **Solve mode blur/click race**: Fixed `solveAutoCheck` — no longer calls `solveCheck` for correct answers, preventing the Delete button race that cleared correct synonym answers
- **Continue solving**: `cordelia_active_solves` in localStorage tracks in-progress puzzles. Home page shows "Continue solving" section with progress and direct links
- **Tools tab persistence**: Pattern/anagram/similar results cached when switching tabs within same clue
- **Anagram clear on close**: `closeToolsOverlay` now calls `anagramClear()` so fodder doesn't carry between clues
- **Floating widget removed**: From base.html (now `{% block helper_widget %}` opt-in). Puzzle, puzzles, puzzle_list pages override to empty

### Learn Zone Redesign (in progress)
- **Full spec saved**: `documents/learn_zone_redesign_spec.md`
- **Key principles**: Wordplay types first, simple explanations in Cordelia's voice, no separate practice mode, tutorial uses real puzzle with real tools
- **Tutorial approach decided**: Use a real partially-filled puzzle (not the 5x5 mini grid). User picks a HIGH-quality puzzle, pre-fills 60-70% of answers, leaves a mix of unsolved clue types. Everything uses existing infrastructure — no special code needed.
- **5x5 grid built but being replaced**: FEAST/EARTH/ARGUE/STUFF/THEFT word square exists in DB (source=cordelia, puzzle_number=1) with guided walkthrough state machine. BUT crossing propagation issues and custom hook complexity led to decision to use real puzzle instead.
- **Tutorial Cordelia tips**: Step-by-step walkthrough for first clue (click word → indicators → Tools → anagram solver → Check → Add to grid → Show grid). Persistent companion for remaining clues with type-specific hints and answer explanations. Draggable tips. Back button.
- **Next**: User to pick a real puzzle candidate for the tutorial
